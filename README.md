# 🌐 Portafolio Web - Rubén Eufracio

Bienvenido a mi portafolio web 👋
Este proyecto es una página web de presentación personal desarrollada con fines visuales, donde muestro mi estilo de diseño y estructura como desarrollador frontend.

---

## 🎨 Sobre el proyecto

Este portafolio está enfocado principalmente en el diseño visual y la experiencia de usuario.
No cuenta con funcionalidades avanzadas de backend, ya que su objetivo es representar de forma clara y atractiva mi perfil profesional.

---

## 🛠️ Tecnologías utilizadas

* HTML5
* CSS3
* JavaScript (básico para efectos visuales)

---

## ✨ Características

* Diseño moderno y minimalista
* Animaciones suaves
* Sección de presentación personal
* Enlaces a redes sociales

---

## 🎯 Objetivo

El propósito de este proyecto es:

* Mostrar mis habilidades en maquetación web
* Practicar diseño UI/UX
* Tener una presentación profesional en línea

---

## 📂 Estructura del proyecto

* `index.html` → Página principal
* `style.css` → Estilos del sitio
* `script.js` → Efectos visuales
* `img/` → Recursos gráficos
